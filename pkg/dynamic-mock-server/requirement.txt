response Func template "Concept": run one by one : you can add more features if it good. please edit based on what best

group: PrepareData
- IfRequestHeader(HeaderName, Condition, Value, DynamicVariable, toBeValue)
    example: IfRequestHeader("Authorization", dynamicMock.ConditionEqual, "Bearer 123", "VAR_A", 123.435)
- IfRequestJsonBody(Field, Condition, Value, DynamicVariable, toBeValue)
    Field for example:  a.b.c, a[0].b.c
    dynamicVariable keep in map[string]interface{} lifetime on each request
- IfRequestPath(Condition, Value, DynamicVariable, toBeValue)
- IfRequestQuery(Field, Condition, Value, DynamicVariable, toBeValue) // in case of url has query params

group: Generator
- GenerateRandomString(length, toDynamicVariable)
- GenerateRandomInt(min, max, toDynamicVariable)
- GenerateRandomIntFixLength(length, toDynamicVariable)
- GenerateRandomDecimal(min, max, max_decimal, toDynamicVariable)
- HashedString(fromDynamicVariable, HashAlgorithm, toDynamicVariable)

group: DynamicVariable
- ConvertToString(dynamicVariable)
- ConvertToInt(dynamicVariable)
- Delete(dynamicVariable)

group: SetupResponse
- SetJsonBody(jsonBody)
   example with dynamic variable: SetJsonBody(`{"a": {{.VAR_A}}}`)  // please try to find best way to write template inside data, in this sample is {{}} but if there is betterway please use it
- SetStatusCode(statusCode)
- SetWait(timeout)

- SetMethod(method)
- SetHeader(key, value)
